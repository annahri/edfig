# edfig
Edit your config files anywhere.

## Usage
```
Usage: 
  edfig [subcommand] [config name]
  edfig [config name]
       
Example:
  edfig add vim "$HOME/.vimrc"
  edfig vim
  edfig edit vim
  edfig edit
  edfig ls

Subcommands:
  add    Add new config file to list
  rm     Remove config file from list
  edit   Edit an entry
  ls     List all stored configs
  help   Print this
```
